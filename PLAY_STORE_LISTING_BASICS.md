# Play Store Listing Basics

## App identity

- App name: `Idara-Mohalla Attendance Mobile`
- Package name: `com.idaramohalla.attendance`
- Current version: `1.0.0`
- Current Android version code: `1`

## Positioning

This is the phase-1 mobile app for the existing attendance system. The first release covers:

- Dashboard
- Record Attendance
- Mobile sign-in connected to the current PHP/MySQL system

## Short description

Use this in Google Play's short description field:

```text
Dashboard and attendance recording for Idara-Mohalla staff.
```

## Full description

Use this as the first store listing draft:

```text
Idara-Mohalla Attendance Mobile helps authorized staff view their dashboard and record attendance quickly from Android.

Phase 1 includes:
- Mobile dashboard for the current role and scope
- Fast attendance recording by ITS ID
- Role-based access tied to Idara and Mohalla
- Direct connection to the existing attendance system

This release is designed for operational use by authorized users of the attendance platform. Access depends on the role and scope already assigned in the main system.
```
